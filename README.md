# Los Creativos - Claude Design Skill

Este repositorio contiene las habilidades (skills) y configuraciones personalizadas para optimizar el flujo de trabajo de diseño y desarrollo web utilizando Claude AI dentro del ecosistema de **Los Creativos**.

## 🚀 Contenido del Repositorio

- **`claude-project/`**: Configuraciones de proyecto, tokens de diseño y guías de estilo para asegurar la consistencia visual.
- **`skills/`**: Definiciones de habilidades específicas para Claude:
  - `loscreativos-claude-design`: Optimizaciones para diseño UI/UX.
  - `loscreativos-claude-ai`: Lógica y prompts avanzados.
  - `loscreativos-claude-code`: Patrones de desarrollo y buenas prácticas.

## 🛠️ Cómo Usar

Estas habilidades están diseñadas para ser importadas o referenciadas por Claude para:
1. Mantener la consistencia de marca en todos los desarrollos.
2. Acelerar la creación de componentes UI siguiendo el `ui-kit.css`.
3. Validar que el código generado cumpla con los estándares de Los Creativos.
